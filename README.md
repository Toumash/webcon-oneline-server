# Webcon DEV Vagrant Box
> Vagrant webcon provisioning template for developers.

**WORK IN PROGRESS**

# How to run?

1. Install prerequisites
```
choco install ruby vagrant
``` 
2. Run from elevated powershell
```
vagrant up --provision
```
